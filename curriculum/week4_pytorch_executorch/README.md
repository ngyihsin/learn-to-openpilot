# Week 4 — PyTorch / ExecuTorch

Train a model, then ship it to run on-device the way openpilot does: tensors & autograd,
training loops, CNNs, exporting, ExecuTorch inference, and quantization.

> Needs `torch`. CPU-only is fine: `pip install torch --index-url https://download.pytorch.org/whl/cpu`
> ExecuTorch (Day 27) installs separately — see its lesson.
