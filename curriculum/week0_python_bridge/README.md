# Week 0 — The Python Bridge

> *Goal: one lesson that unlocks all the others. The curriculum's homework files use a handful of
> Python features — classes, dunder methods, exceptions, decorators, type hints — that basic Python
> tutorials often skip. Learn them once here, and every later file reads naturally.*

| Day | Topic | Format | You'll be able to | Status |
|-----|-------|--------|-------------------|--------|
| 00 | **The Python this course is written in** | notebook-free, pytest | Read & write classes, dunders, exceptions; ignore type hints; reason with Big-O | ✅ |

**Who should do this:** anyone who has basic Python (variables, functions, lists, loops, `if`) but has
never written a `class`, never used `raise`/`try`, or doesn't know what `O(n)` means. That's exactly
the student Day 01 assumes — so do Day 00 first.
